# CS Quality Review
## Metrics to Track
- ⏱️ Response time trung bình
- ✅ Resolution rate (% câu hỏi được giải quyết)
- 🔄 CS→Order conversion rate
- 😤 Negative sentiment count
- 📊 Total conversations
## Steps
1. Query conversation data from Gateway
2. Analyze response quality: accuracy, tone, helpfulness
3. Identify patterns: common questions, repeated issues
4. Generate recommendations
